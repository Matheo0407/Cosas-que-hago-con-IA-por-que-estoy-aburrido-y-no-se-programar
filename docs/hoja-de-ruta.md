# Hoja de ruta

## Que existe ahora

- una base jugable inspirada en el genero
- flujo de menus
- datos de un nivel demo
- conceptos de editor
- conceptos de publicacion y comunidad local
- ports en Java, HTML, Python, C++ y Lua

## Mejor orden para continuar

1. Convertir el port de Java en la implementacion de referencia.
2. Extraer archivos de nivel compartidos y ayudas de serializacion.
3. Anadir pruebas deterministas de colision para ajustar las fisicas.
4. Anadir sincronizacion musical y marcadores de ritmo.
5. Anadir mas objetos de juego y efectos de camara.
6. Construir un servicio online pequeno para cuentas y compartir niveles.
7. Reemplazar los visuales provisionales por arte original de produccion.

## Trabajo necesario para acercarse mas

- simulacion de paso fijo y validacion de repeticiones
- sistemas de pendientes, portales, gravedad y cambios de velocidad
- editor mas completo con seleccion, copiar/pegar, deshacer y capas
- backend real para subidas, descargas, valoraciones y moderacion
- licencias de audio y pipeline correcto de assets
- entrada movil y empaquetado

## Limite importante

El proyecto debe seguir siendo `inspirado por` el genero, no una reproduccion literal del contenido protegido de otro juego comercial.
