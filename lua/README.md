# Port en Lua

Este port apunta a Love2D 11.x.

Ejecutalo despues de instalar Love2D:

```bash
love "Geometry Jump Lua Edition"
```

El script incluye:

- menu
- seleccion de nivel
- jugabilidad principal
- editor
- comunidad local simulada

Punto de entrada del proyecto:

- `Geometry Jump Lua Edition/main.lua`
